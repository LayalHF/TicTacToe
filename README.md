# React-Prototype-TicTacToe
A simple browser game TicTacToe can be played with two players.
Prototype built with the following technologies:
- React v19.2
- JavaScript
- Tailwind CSS v4

Notable features include:
- fully responsive, and scales proportionally to small  devices.
- Two-player rounds

Setup:
  - Vite build tool
  - used npm to install/update/manage modules
